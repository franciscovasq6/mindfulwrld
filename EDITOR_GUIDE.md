# Content Editor Guide - mindfulwrld.com

Welcome! This guide will help you create and manage blog posts on mindfulwrld.com.

## Logging In

1. Go to **https://mindfulwrld.com/admin**
2. Click **"Login with Netlify Identity"**
3. Enter your email and password
4. Click **"Log in"**

## Creating a New Blog Post

### Step 1: Start a New Post

1. Click **"New Blog Posts"** button (top right)
2. You'll see an empty form

### Step 2: Fill in the Details

#### **Title** (Required)
- Keep it clear and engaging
- Under 60 characters works best for SEO
- Example: "5 Morning Rituals for a Mindful Day"

#### **Description** (Required)
- A brief summary of your post (1-2 sentences)
- 120-160 characters is ideal
- This appears in search results and on blog cards
- Example: "Start your day with intention using these simple mindfulness practices."

#### **Author** (Auto-filled)
- Usually set to "mindfulwrld"
- You can change it to your name if you prefer

#### **Publish Date** (Required)
- Click the calendar icon
- Select the date and time you want the post published
- For immediate publishing: select current date/time
- For scheduling: select future date/time

#### **Hero Image** (Optional but recommended)
- Click **"Choose an image"**
- Upload from your computer OR select from media library
- **Best practices:**
  - Size: 1200px x 630px (landscape)
  - Format: JPG or PNG
  - File size: Under 500KB
  - Relevant to your content

#### **Tags** (Optional)
- Click **"Add tags"**
- Enter relevant keywords (press Enter after each)
- Use 3-5 tags per post
- Examples: mindfulness, meditation, wellness, stress-relief
- Use existing tags when possible for consistency

#### **Body** (Required)
This is where you write your main content!

### Step 3: Writing Your Content

The editor supports **Markdown** formatting. Here's how:

#### Headers
```
# Main Title (H1 - use sparingly)
## Section Header (H2)
### Subsection (H3)
```

#### Text Formatting
- **Bold**: Select text ‚Üí Click **B** button (or use `**text**`)
- *Italic*: Select text ‚Üí Click *I* button (or use `*text*`)
- Links: Select text ‚Üí Click link icon ‚Üí Paste URL

#### Lists

**Bullet points:**
```
- First item
- Second item
- Third item
```

**Numbered lists:**
```
1. First step
2. Second step
3. Third step
```

#### Adding Images in Content

1. Place cursor where you want the image
2. Click image icon
3. Upload or select image
4. Add alt text (describes the image)
5. Click "Insert"

#### Quotes
```
> This is a quote. Use it for emphasis or to highlight important points.
```

#### Code (if needed)
```
`inline code` for short snippets
```

### Step 4: Preview Your Post

1. Click **"Show preview"** (top right toggle)
2. Review how your post looks
3. Click again to return to edit mode

### Step 5: Save or Publish

You have three options:

#### Save as Draft
- Click **"Save"** button
- Post is saved but NOT published
- Only you can see it
- Come back anytime to edit

#### Submit for Review
1. Click **"Set status"** dropdown
2. Select **"In review"**
3. Other editor will review and can approve
4. They'll receive a notification

#### Publish Immediately
1. Click **"Publish"** dropdown
2. Select **"Publish now"**
3. Post goes live in ~2 minutes!
4. You'll receive a confirmation email

## Editing Existing Posts

1. From dashboard, click **"Blog Posts"** in sidebar
2. Find the post you want to edit
3. Click on it to open
4. Make your changes
5. Click **"Save"** or **"Publish"**

## Publishing Workflow

We use a **3-stage workflow**:

1. **Draft** ‚Üí Working on the post, not visible
2. **In Review** ‚Üí Ready for teammate to check
3. **Published** ‚Üí Live on the website

### To Move a Post Through Workflow:

- **Draft ‚Üí In Review**: Click "Set status" ‚Üí "In review"
- **In Review ‚Üí Published**: Reviewer clicks "Publish" ‚Üí "Publish now"
- **Back to Draft**: Click "Set status" ‚Üí "Draft"

## Best Practices

### Content Guidelines

‚úÖ **DO:**
- Write in a friendly, conversational tone
- Use short paragraphs (2-4 sentences)
- Include headers to break up text
- Add relevant images
- Proofread before publishing
- Use tags consistently

‚ùå **DON'T:**
- Write walls of text (use headers!)
- Forget alt text on images
- Use all caps (looks like shouting)
- Publish without preview
- Skip the description field

### SEO Tips

1. **Titles**: Include main keyword, keep under 60 characters
2. **Descriptions**: Summarize post in 120-160 characters
3. **Headers**: Use H2 and H3 to organize content
4. **Links**: Link to other relevant posts when appropriate
5. **Images**: Always add descriptive alt text

### Image Guidelines

**Hero Images:**
- Landscape orientation (16:9 ratio)
- 1200px x 630px ideal
- Under 500KB file size
- JPG for photos, PNG for graphics

**In-Content Images:**
- Relevant and high-quality
- Optimized for web (compress before upload)
- Include descriptive alt text

## Troubleshooting

### Can't log in?
- Check your email and password
- Try "Forgot password" link
- Clear browser cache
- Contact admin if issue persists

### Image won't upload?
- Check file size (max 10MB)
- Use JPG or PNG format
- Try compressing the image
- Check your internet connection

### Post not appearing on site?
- Ensure status is "Published" (not Draft or In Review)
- Wait 2-3 minutes for build to complete
- Clear your browser cache
- Check the /blog page

### Changes not showing?
- Click "Publish" (not just "Save")
- Wait for build to complete (~2 min)
- Hard refresh browser (Cmd+Shift+R on Mac, Ctrl+Shift+R on Windows)

### Lost my work?
- CMS auto-saves every few seconds
- Check "Drafts" in sidebar
- If truly lost, contact admin (we have backups)

## Writing Tips

### Post Structure

A great blog post includes:

1. **Hook** - Grab attention in first paragraph
2. **Problem** - What challenge does this address?
3. **Solution** - Your tips, advice, or insights
4. **Action** - What should reader do next?
5. **Conclusion** - Wrap up with encouragement

### Example Outline

```
Title: "5 Quick Mindfulness Exercises for Busy Days"

Introduction:
- Hook: "Feeling overwhelmed? You're not alone..."
- Promise: "These 5 exercises take under 5 minutes"

Main Content:
- Exercise 1: [Name]
  - How to do it
  - Why it works
  - When to use it

- Exercise 2-5: [Same structure]

Conclusion:
- Recap benefits
- Encourage action
- Invite comments/questions
```
